Installation Notes:

Unpack this Product into your Zope/Plone Products directory. This
product has a number of dependencies.

Product                         Version    URL
-----                           --------   -----------------------------------
Plone                           2.0.4+     http://svn.plone.org/plone/CMFPlone/trunk/
Archetypes                      1.3+       http://svn.plone.org/archetypes/Archetypes/branches/release-1_3-branch/
ATReferenceBrowserWidget        N/A        http://svn.plone.org/archetypes/ATReferenceBrowserWidget/trunk/
AdvancedQuery                   0.6        http://www.dieter.handshake.de/pyprojects/zope/AdvancedQuery.tgz
ExtendedPathIndex               1.0        http://www.plonesolutions.com/products/ploneportlets/0.3beta/tarball/downloadableFile/at_download

(optional)
kupu                            1.1+       http://codespeak.net/svn/kupu/trunk/kupu

Make sure these are all installed and then on a *nix system from the
wicked directory you should be able to type

# make


if you wish to run the tests without using make (which is fine)

# python tests/runalltests.py
