========
 wicked
========

more documentation can be found in the 'docs' directory.

REQUIRES
========

Python 2.4+
Zope 2.9+
Archetypes 1.4+
Plone 2.1+
filter 0.1.7


QUICKSTART
==========

Place 'filter' and 'wicked' into a Plone ready Products directory. Use
Plone quickinstaller to install wicked example types.
